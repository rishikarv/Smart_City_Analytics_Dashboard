# 🌆 Smart City Analytics System

## 📌 Overview

The Smart City Analytics System is an AI-powered platform designed to improve urban management using data-driven insights. It integrates multiple modules to analyze and predict key city parameters.

---

## 🚀 Features

### ⚡ Energy Consumption Prediction

Predicts energy usage (kWh) based on time, temperature, and weekday/weekend patterns.

### 🚦 Traffic Congestion Prediction

Analyzes traffic conditions using vehicle count, speed, road occupancy, and time of day.

### 🚓 Crime Hotspot Detection

Identifies high-risk areas using clustering techniques and visualizes them on a map.

### 🌫️ Air Quality Prediction

Predicts Air Quality Index (AQI) using environmental parameters like PM2.5, PM10, NO₂, CO, temperature, and humidity.

### 🗑️ Waste Route Optimization

Optimizes garbage collection routes by identifying high-fill bins and generating efficient paths.

## ▶️ How to Run

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Run the app:

```
streamlit run app.py
```
